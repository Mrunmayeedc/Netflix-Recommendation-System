# Netflix Recommendation System

A Machine Learning-based Netflix Recommendation System that suggests similar movies and TV shows using **Content-Based Filtering**. The recommendation engine uses **TF-IDF Vectorization** and **Cosine Similarity** to identify similar content based on metadata such as directors, cast, genres, and descriptions.

## Project Overview
This project demonstrates how recommendation systems work in streaming platforms like Netflix.
The system analyzes the content information of movies and TV shows and recommends titles that are similar to the selected movie.
The recommendation is generated using:
- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Filtering
